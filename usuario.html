Consultar Usuario por ID POST

http://smartcityhyo.tk/api/Usuario/Consulta_Usuario_ID.php
{
    "ID_Usuario": "1"
}

Consultar Usuario por ID GET

http://smartcityhyo.tk/api/Consultar_Usuario.php?ID_Usuario=1
Ingresar por parametro: ID_Usuario=?

Consultar Usuario por Email POST

http://smartcityhyo.tk/api/Usuario/Consulta_Usuario_Email.php
{
    "US_Email": "juan@gmail.com"
}

Consultar Usuario por Email GET

http://smartcityhyo.tk/api/buscar_Email.php?US_Email=juan@gmail.com
Ingresar por parametro: US_Email=xxx@gmail.com

Listar Usuario GET

http://smartcityhyo.tk/api/Lista_Usuario.php
No se ingresan parametros
