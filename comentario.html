Web API's

Tip 1:

Usar los links proporcionados en el documento/Utilizar los cuerpos de documento para el método POST

Tip 2:

Para los servicios con fotografía, el formato de envio en postman es desde la sección body>>form-data. Después de ingresar, se debe especificar el campo de la fotografia con un tipo de dato FILE

Agregar Comentario Obra POST

http://smartcityhyo.tk/api/ComentarioObra/insertar_comentario.php 
{
    "ID_Obra": 1,
    "COM_Obra": "Buena Obra",
    "COM_Calificacion": 4,
    "ID_Usuario": 13,
    "COM_Foto": FILE
    
}

Consultar Comentario por IDGET

http://smartcityhyo.tk/api/ComentarioObra/Consultar_Comentario_ID.php? 
Ingresar por parametro: ID_Comentario=?

Consultar Comentarios por UsuarioGET

http://smartcityhyo.tk/api/ComentarioObra/Consultar_Comentario_User.php 
Ingresar por parametro: ID_Usuario=?

Consultar comentarios por ObraGET

http://smartcityhyo.tk/api/ComentarioObra/Consultar_Comentario_Obra.php 
Ingresar por parametro: ID_Obra=?

Listar Comentarios por ID GET

http://smartcityhyo.tk/api/ComentarioObra/listar_comentario_por_id_obra.php 
Ingresar por parametro: ID_Obra=?

Listar Comentarios GET

http://smartcityhyo.tk/api/ComentarioObra/listar_comentario.php 
No se ingresan parametros

Update Comentario POST

http://smartcityhyo.tk/api/ComentarioObra/Update_Comentario.php 
{
    "ID_Comentario": 3,
    "ID_Obra": 10,
    "COM_Obra": "Huancayo",
    "COM_Calificacion": "Prueba",
    "ID_Usuario": 13
}

Eliminar Comentario Obra POST

http://smartcityhyo.tk/api/ComentarioObra/Eliminar_Comentario_Obra.php 
{
    "ID_Comentario": "3"
}
